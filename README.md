# Chocolate Sales Dashboard

## Project Overview
This project presents an interactive sales dashboard built using Power BI to analyze chocolate sales data across different countries, products, and salespersons.

## Objectives
- Track total sales and shipment volume
- Compare performance against sales targets
- Identify top-performing products and regions
- Analyze monthly sales trends
- Segment high vs normal performance categories

## Tools Used
- Power BI (Data Visualization, DAX)
- Excel (Data Cleaning & Preparation)

## Key Features
- KPI Cards for Total Sales, Boxes Shipped, and Sales Targets
- Trend indicators showing performance changes over time
- Monthly sales trend analysis
- Country-wise sales visualization (Map)
- Product performance breakdown
- Dynamic filters (Date, Salesperson, Product)

## Key Insights
- Top 3 products contribute majority of revenue
- High-performance category generates ~25% of total sales
- Sales show strong fluctuations across months indicating seasonal trends
